## 📌 Features

| Feature | Description |
|---------|-------------|
| 🔍 TCP Port Scan | Multi-threaded, fast scanning |
| 📡 UDP Scan | Common UDP ports detection |
| 🌍 GeoIP + ASN | IP location, ISP, ASN info |
| 🔒 SSL/TLS Info | Certificate details |
| 🌐 HTTP Headers | Server response headers |
| 🖥️ OS Detection | TTL-based OS fingerprinting |
| 📋 DNS Lookup | A, AAAA, PTR, FQDN records |
| 🔎 Whois Lookup | Domain registration info |
| 📡 Ping Sweep | Find live hosts in network |
| 🗺️ Traceroute | Network path tracing |
| 📊 Export | HTML, JSON, TXT reports |
| 🕓 Scan History | Last 20 scans saved |

## ⚙️ Installation

```bash
# Clone karo
git clone https://github.com/YOURUSERNAME/HBScan.git

# Folder mein jao
cd HBScan

# Run karo
python3 hbscan.py
```

## 🚀 Usage

```bash
python3 hbscan.py
```

Menu se option select karo:
- **[1]** Full Scan
- **[2]** Ping Sweep
- **[3]** DNS Lookup
- **[4]** Traceroute
- **[5]** GeoIP + ASN
- **[6]** SSL/TLS Info
- **[7]** HTTP Headers
- **[8]** Whois Lookup
- **[9]** Scan History
